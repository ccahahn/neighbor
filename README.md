# Neighbor

Admin-facing prototype that helps school administrators reduce chronic absenteeism by surfacing opted-in families who could walk kids to school together.

**[Try the prototype →](https://neighbor-psi.vercel.app/)**

Map-based candidate selection with layered consent: registration opt-in (generic), text confirmation (specific to a pairing), and a phone call from the admin to the family who needs help. The system surfaces data; the human decides.

Built by Cecilia Cahahn.
